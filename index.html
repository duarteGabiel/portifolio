Um site de vendas de ingresso do Batman desenvolvido com HTML e CSS onde é possível ver toda a temática do filme Batman, assim como seu trailer e os principais atores nele foi meu primeiro contato com programação sem ter muito experiencia tentei fazer uma página de contato Onde você inseria seus dados para enviar ao vendedor.

Esse é um clone do YouTube, o front-end feito com react.js e TypeScript permite uma experiência idêntica ao do YouTube, pois tem sua API de vídeos integradas podendo fazer pesquisa de quaisquer vídeos que estejam no YouTube, você também pode criar uma conta e criar os seus vídeos nesse clone o back-end por sua vez feito com node.js tendo um banco de dados integrado possibilitando a criação e armazenamento de login e vídeos no site, assim você pode Sempre estar logado para acessar suas informações e vídeos a qualquer hora.

Esse é um clone do WhatsApp web, bem simples, o front-end feito com React e JavaScript, tendo um visual bem parecido e o back-end feito com node.js e socket.io permitindo a interação das pessoas em um grupo de Conversa onde o chat em tempo real retorna as mensagens quase que instantâneas.
